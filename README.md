# 말해보카식 복습

TOEFL 단어 암기용 **단일 파일 웹앱**. 빌드·의존성·서버 없음.

## 실행
`index.html`을 브라우저로 열면 끝.

## 배포
비공개 레포라 **Netlify로 배포**한다(무료 · HTTPS). 앱 파일만 올리면 된다(`index.html` · `manifest.json` · `sw.js` · `icons/`).
서비스워커 · 동기화 · 홈 화면 설치는 **HTTPS에서만** 켜지므로, `file://`가 아니라 배포 URL로 열어야 앱 기능이 동작한다.

- **라이브: https://toeflvoca.netlify.app**
- 재배포: 레포 루트에서 `bash deploy.sh` (Netlify CLI 직접 배포 — 빌드 없음 = 무료 빌드분 0 소모). 토큰은 레포 밖 `~/.config/toeflvoca/netlify_token`에 있다.
- ⚠️ 깃 자동배포가 아니다. `git push`는 소스만 올리고, **라이브 반영은 `deploy.sh` 실행**이 따로 필요하다.

## 앱으로 쓰기 (PWA)
배포 URL을 폰 브라우저로 열고 **"홈 화면에 추가"** 하면 전체화면 앱처럼 설치된다(아이폰 · 안드로이드 · 데스크톱). 오프라인에서도 열리고, 코드를 고쳐 `git push` 하면 온라인에서 자동으로 최신 버전이 뜬다(앱 재설치 불필요).

## 기기 간 동기화 (로그인 없음)
학습 기록은 브라우저 `localStorage`에 저장되고, 동시에 클라우드(Supabase)에 백업된다. 로그인은 없다 — 첫 실행 때 기기마다 **동기화 코드**가 생기고, 다른 기기에서 시작 화면의 "기기 간 동기화 · 백업"에 그 코드를 넣으면 진도가 이어진다. 구조는 [`CLAUDE.md`](./CLAUDE.md) 11장 참고.

## 구성
세 묶음의 카드로 자기주도 복습을 한다.
- **문장 복습** — 한국어 문장의 빈칸을 영어로 채우기
- **단어 암기 · TOEFL Voca 01** (레슨 1-1 / 1-2 / 2-1)
- **해커스 보카 Day 1** (1–20 / 21–40 / 41–57)

단어 암기·해커스 카드는 *예문 빈칸 채우기 → 동의어 4지선다* 2단계로 푼다.
학습 기록은 브라우저에 저장되고, 틀린 카드만 모아 다시 볼 수 있다.

세트를 고르고 **시작하기**를 누르면 먼저 단어 목록을 훑어본 뒤 퀴즈로 들어간다.
